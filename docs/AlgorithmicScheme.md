1. Загрузка конфигурации (CLI + YAML) и целей.
2. Проверка доступности Masscan (shutil.which).
3. Если Masscan доступен и не задан --force-nmap:
   - Запуск Masscan с таймаутом 30 сек, парсинг JSON.
4. Если Masscan не дал результатов или пропущен:
   - Разбор диапазона портов в список.
   - Запуск Nmap для КАЖДОГО IP из целей (параллельно, ThreadPoolExecutor).
   - Фильтрация результатов: оставляем только state == 'open'.
5. Если Masscan сработал:
   - Группировка открытых портов по IP.
   - Запуск Nmap только для этих IP и портов (с версиями -sV).
   - Фильтрация open.
6. Обогащение веб-сервисов (HTTP/HTTPS) для портов 80,443,8080,8443.
7. Сохранение в SQLite (scans, open_ports, port_history).
8. Проверка на высокорисковые сервисы (по портам или имени).
9. Отправка уведомлений в Telegram/Email/Webhook при обнаружении.
10. Вывод сводки в лог.
